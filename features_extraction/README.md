# Replicability attempt

This folder contains a Python re‑implementation of the SAS feature‑extraction code used to generate the 94 features in Gu et al. (2020), “Empirical Asset Pricing using Machine Learning.”

The original SAS code was developed by ![Jeremiah Green](https://sites.google.com/site/jeremiahrgreenacctg/home). It pulls Compustat, CRSP, and IBES data via WRDS.  

My goal was to centralize the feature‑extraction logic in Python. However, differences in data‑source access across users and institutional WRDS subscriptions created some challenges.  

Further explanation is available in my master’s thesis.
